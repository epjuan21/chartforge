# 📊 ChartForge — Generador de Gráficos Estadísticos

## Resumen del Proyecto

**ChartForge** es una aplicación web construida con **Next.js** que permite generar gráficos estadísticos de manera rápida, atractiva y configurable. Está diseñada para usuarios que necesitan crear gráficos profesionales para presentaciones, informes o documentos sin necesidad de herramientas complejas ni registro.

---

## 🎯 Objetivos Principales

| # | Objetivo | Descripción |
|---|----------|-------------|
| 1 | **Generación rápida** | Permitir crear gráficos en segundos ingresando datos directamente |
| 2 | **Variedad de tipos** | Soportar gráficos de barras, líneas, anillos/dona, torta, área, radar y mixtos |
| 3 | **Personalización visual** | Configurar colores, fuentes, tamaños, leyendas y estilos |
| 4 | **Exportación** | Descargar gráficos como PNG, SVG y PDF listos para usar |
| 5 | **Sin fricción** | No requiere autenticación, carga de archivos ni persistencia de datos |
| 6 | **Responsive** | Funcionar correctamente en desktop y tablets |
